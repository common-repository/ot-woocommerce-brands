Add Brands taxonomy for products for WooCommerce plugin. Show brand name on product pages and category pages. Use widgets to display brands list.

== Description ==
This plugin allow you to easy add brands to your WooCommerce products on your existing site (with any WordPress theme) and include additional features like:
*   Create unlimited Brands categories for your products (you can assign multiple brands to your one product)
*   Display brand name on Product page (choose from 8 positions to display in Plugin Admin Panel)
*   Display brand name on Category pages and any Product listing pages (Choose from 5 positions to display in Plugin Admin Panel)
*   Filter product by Brands on specials brands pages (every brand on your site will have its own page with assigned products)
*   WordPress Widget to show your brands list in Sidebars in different ways
*   Plugin work with ANY theme that support WooCommerce
*   100% Responsive all plugin elements and pages
*   Translation ready (PO/MO files included)

== Installation ==
* Download to your plugin directory or simply install via Wordpress admin interface.
* Activate.
